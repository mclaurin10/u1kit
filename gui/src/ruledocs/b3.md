## B3 — BBL-specific filament fields

Removes `filament_extruder_variant`, `inherits` chains referencing Bambu profiles, and `compatible_printers` entries pointing to non-U1 printers. Shared logic with the profile-sanitizer tool.

**Severity:** warn — these fields are usually harmless but can cause profile-resolution surprises inside Snapmaker Orca.

**Why this matters:** an `inherits: "Generic PLA @BBL X1C"` chain, if unresolved, can silently fall back to slicer defaults. Cleaning these up makes the output's behavior predictable.
