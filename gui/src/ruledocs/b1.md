## B1 — Filament count > 4

The U1 is a four-tool toolchanger; files defining more than four filaments must be consolidated. Auto-fix is deliberately conservative: it refuses in auto mode (requires explicit user intent) and proposes merges in interactive mode by CIEDE2000 color distance (closest pair first).

**Severity:** fail — Snapmaker Orca rejects files that exceed the four-slot cap.

**Why this matters:** merging filaments reassigns every per-filament field and remaps selector scalars. Done wrong, you silently print a wall in the wrong color. The interactive path shows each proposed merge with a diff and lets you veto it.
