## B2 — Filament-to-tool mapping missing or invalid

Ensures each used filament has a `filament_map` entry with an extruder index in 1–4. If missing (common on U1 native exports) or out of range, auto-fix assigns by first-use order.

**Severity:** fail — Snapmaker Orca's Print Preprocessing rejects files without a valid mapping.

**Why this matters:** on the U1, the filament slot IS the extruder — there's no remapping at load-time like an AMS. Missing mapping = preprocessor can't tell the firmware which tool to use when.
