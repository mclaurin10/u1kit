## B4 — Per-filament speed overrides missing for flexibles

If a filament's `filament_type` is TPU/PEBA and it lacks explicit per-filament speed caps, auto-fix injects conservative overrides (outer wall, inner wall, infill) derived from `filament_max_volumetric_speed × line cross-section × 0.8`.

**Severity:** warn — the slicer's global speeds often exceed what flexibles can extrude.

**Why this matters:** under-extrusion on flexibles usually looks like "the print works but drops detail" — the filament spools through but the hot end can't keep up. Belt-and-suspenders speed caps make the same file print reliably across multiple machines.
