## D1 — Mixed-filament height bounds below layer_height

Detects when Full Spectrum's `mixed_filament_height_lower_bound` is less than `layer_height`. Auto-fix locks all three values (`layer_height`, `mixed_filament_height_lower_bound`, `mixed_filament_height_upper_bound`) to the same uniform value. Default 0.2 mm; tunable via `--uniform-height`.

**Severity:** fail — this is the single biggest repeated failure mode in Full Spectrum-to-U1 workflows.

**Why this matters:** the `mixed_filament_height_lower_bound` defaults to 0.04 mm in some Full Spectrum exports. On a 0.2 mm layer height, that's a 37-layer Z-hop per toolchange — enough to crash the head into any tall feature. Locking all three to one value gives you a clean single-height print with predictable toolchange moves.
